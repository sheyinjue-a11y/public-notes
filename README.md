### 弗洛伊德(没看过书籍作品，仅依据ai的摘要总结，由Gemini补充)
## 原因论：过去（创伤或未满足的本能驱力）-> 产生心理防御机制 -> 压抑进潜意识 -> 现在的行为和想法
## 路径：觉察 -> 识别并卸下无效的防御机制 -> 理解与接纳 -> 潜意识意识化
## 目标：通过理解潜意识接纳自我，将潜意识意识化，从而更好地通过【自我】（意识理性）去协调【本我】（原始欲望）与【超我】（道德规训）之间的冲突。客观上的目的是拿回意识的主权以改造现在。

### </br></br></br>阿德勒（依据《被讨厌的勇气》，由Gemini补充）
## 目的论：现在的【目的】（包括显性和不易觉察的隐性）-> 现在做出的选择（宏观上是一种生活方式）-> 当下的情绪和行为
## 路径：认识到【目的】支配我们当下的选择  -> 改变【目的】（具体方法包括但不限于进行“课题分离”（分清自己与他人的课题），重新为过去赋予意义，正确利用自卑感与追求优越等） -> 主动掌控现在的想法和行动
## 目标：通过认识到过去的意义由现在赋予，摆脱【原因论】的“舒适圈”。进而认识到一切烦恼皆源于人际关系，但“目的”由自己选择；通过课题分离走出他人的期待，最终建立“共同体感觉”，主动掌控现在的想法和行动。

### </br></br></br>共同点：
分析理论的基础和起点不同（原因论偏向决定论，目的论强调当下的主观能动性）。但殊途同归，最终都是为了帮助人们接纳自我（与过去），重新让【自己】掌控现在的行为和想法，夺回当下的控制权
